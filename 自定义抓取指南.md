# Horizon 自定义抓取指南

编辑文件：`data/config.github.json`（GitHub Actions 用这个）  
本地测试：改完后复制为 `data/config.json`，运行 `uv run horizon --hours 24`

---

## 一、改 AI 模型

```json
"ai": {
  "provider": "deepseek",
  "model": "deepseek-chat",
  "api_key_env": "DEEPSEEK_API_KEY",
  "languages": ["zh", "en"],
  "analysis_concurrency": 1,
  "throttle_sec": 1
}
```

| provider | 常用 model | Secret 名称 |
|----------|-----------|-------------|
| deepseek | deepseek-chat | `DEEPSEEK_API_KEY` |
| openai | gpt-4o-mini | `OPENAI_API_KEY` |
| gemini | gemini-2.0-flash | `GOOGLE_API_KEY` |
| anthropic | claude-sonnet-4-5 | `ANTHROPIC_API_KEY` |
| ali | qwen-plus | `DASHSCOPE_API_KEY` |

改 `provider` / `model` / `api_key_env` 后，在 GitHub Secrets 里添加对应密钥。

## 二、改信息源（sources）

### RSS 博客

```json
"rss": [
  {
    "name": "你喜欢的博客名",
    "url": "https://example.com/feed.xml",
    "enabled": true,
    "category": "ai-tools"
  }
]
```

找 RSS：在博客页搜 `rss` / `atom` / `feed`，或用 `https://rss.app` 生成。

### Reddit 版块

```json
"reddit": {
  "enabled": true,
  "subreddits": [
    {
      "subreddit": "LocalLLaMA",
      "enabled": true,
      "sort": "hot",
      "time_filter": "day",
      "fetch_limit": 12,
      "min_score": 40
    }
  ],
  "fetch_comments": 8
}
```

### Hacker News

```json
"hackernews": {
  "enabled": true,
  "fetch_top_stories": 25,
  "min_score": 120
}
```

`min_score` 越高越严，只留高分帖。

### GitHub 动态 / Release

```json
"github": [
  {
    "type": "user_events",
    "username": "karpathy",
    "enabled": true
  },
  {
    "type": "repo_releases",
    "owner": "vllm-project",
    "repo": "vllm",
    "enabled": true
  }
]
```

### Telegram 频道（可选）

```json
"telegram": {
  "enabled": true,
  "channels": [
    { "channel": "频道用户名", "enabled": true, "fetch_limit": 20 }
  ]
}
```

## 三、改筛选口味（filtering）

```json
"filtering": {
  "ai_score_threshold": 6.5,
  "time_window_hours": 24,
  "max_items": 15,
  "category_groups": {
    "ai": {
      "limit": 8,
      "categories": ["ai-tools", "machine-learning"]
    }
  },
  "default_group_limit": 5
}
```

| 字段 | 作用 | 调参建议 |
|------|------|----------|
| `ai_score_threshold` | AI 打分门槛（0–10） | 太高→日报空；6.0–7.0 较均衡 |
| `max_items` | 每天最多几条 | 15 适合晨读 |
| `category_groups` | 每类上限 | 防止单一来源霸屏 |
